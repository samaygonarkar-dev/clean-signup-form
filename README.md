# Clean & Responsive Signup Form

A neat, responsive, and modern signup form built to practice clean frontend layouts. This project focuses on solid HTML5 structure and highly polished custom CSS3 without relying on heavy frameworks or bloated libraries.

## 🚀 Features

* **100% Vanilla Tech:** Built purely with semantic HTML5 and custom CSS3.
* **Fully Responsive:** Optimized for a seamless experience across mobile, tablet, and desktop screens.
* **Modern UI/UX:** Features a polished, clean aesthetic with smooth interactive states (hovers, focus rings).
* **Lightweight & Fast:** Zero dependencies, zero compilation steps—just raw, efficient frontend code.

🛠️ How to Use:
Since this project has no build steps or dependencies, you can run it instantly:
1. Clone the repository:
GitBash: 
git clone [https://github.com/samaygonarkar-dev/clean-signup-form.git](https://github.com/samaygonarkar-dev/clean-signup-form.git)

2. Navigate into the directory:
GitBash:
cd clean-signup-form

3. Open the project:
Simply double-click index.html to open it in your favorite web browser, or use an extension like Live Server in VS Code for live-reloading.
